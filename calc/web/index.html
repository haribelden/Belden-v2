Smaller Circles in a Larger Circle

Estimating number of smaller circles that fits in a larger circle - ex. how many pipes or wires fits in a larger pipe or conduit

Sponsored Links

The calculator below can be used to estimate the maximum number of smaller circles that fits into a larger circle. The calculator can be used to calculate

  • the number of smaller pipes that fits in a larger pipe or tube
  • the number of wires possible in a conduit
  • the number of fibers that fits in a connector
  • and similar applications

Input

  • the inside diameter of the large circle (or pipe, tube, conduit, connector), and
  • the outside diameters of the small circles (or pipes, wires, fiber)

The default values are for a 10 inch pipe with 2 inch smaller pipes - dimensions according ANSI Schedule 40 Steel Pipes.

inside diameter of outside larger circle (in, mm, m ..)

outside diameters of inside smaller circles  (in, mm, m ..)

 

 

Calculate Maximum Number of Smaller Circles in Larger Circles - array of multiple smaller and larger circles

In the boxes below

  • add a comma separated list with the sizes of the larger circles
  • add a comma separated list with the sizes of the smaller circles

An array with the maximum number of smaller circles within the larger circles will be created.
